2017-03-06

路还长天很亮，好好走下去。每次走这条路的感觉都还停留在第一次走，哦。是前年这会儿吧……健身回来的路上，以前别人怂恿去健身房的时候，是拒绝的，但是现在挺喜欢这个地方，的确，在这里可以碰到很多有浑身充满正能量的人，这样我的生活变得更充实。周六下午和本科一同学唠了一天，她说现在的我给她的感觉是很乐观，有理想，有目标，知道自己想要什么，，我感觉我能上天飞了，要不是天花板拦着（忘了从谁那学的这句话）但是人总要在不知足与学会知足间徘徊，这样才能不骄不馁吧，毕竟昨天的会议主题还停留在现在对学术的不负责可以直接映射到以后的工作态度呢，然而现在的脸皮是越来越厚了，事不关己的样子，也是越来越自私了，只想好好做毕设，好好提升自己，为找差事准备，这就是本年度规划了。就某天一瞬间的想法，还是选择了好好学英语，可怜了我那由入门到放弃仅仅会几个日常用语的Japanese了，本周总结就这样了，小论文，软著，开题报告，课题规划……啊。。哦呀粟米那塞～好久不提笔都不会作文章的赶脚，果然脑子不用会生锈的。。
