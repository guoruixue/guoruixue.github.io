2017-03-08

今日箴言，对于梦想不要怂，然而我的梦想是什么？我好像并不知道。“新习惯要能满足同样的内在需求，才可替代旧习惯。”不能更同意这句话，说起习惯，就不由得想起社交软件，最先抛弃qq的时候是大一二吧，因为微信和朋友圈，虽然那会儿最在乎的人每天都见，和家人也是电话联系。再到读研以后，qq就是传文件的最佳工具，。但是也会忍不住每天登qq，为了聊天，虽然学姐好奇过这个年代我还在聊qq，第二次抛弃qq是聊天阵地转到微信吧。再后来，删了微信，外加qq。就是现在，看朋友圈和好友动态好像早已不是必须流程，时而心不在焉，时而嗤之以鼻。却会花大把时间浪迹于知乎，浪迹于那个没有人认识我的地方，除了一个匿名好友……。所以……，会不会只是自己在一步一步断自己社交的路，所以，我终究是一个没有原则的人。。。所以，我需要新习惯来打破这该死的旧习惯。
