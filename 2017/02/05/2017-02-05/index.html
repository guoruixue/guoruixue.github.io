2017-02-05

自己养了一年多的抠鼻小猫猫毫无前兆的就屎了，真的是前一天晚上还好好的，说不难受是假的，我又不是不重感情的禽兽。本来今天是和我从小一起长大的姑姑大喜之日，按照习惯昨晚应该在她家，我却因为心情不好提早回家睡大觉，晚上做梦都是猫见我半天没反应在叫我，啊。。。早上起来看到家里群中昨晚的狂欢，总感觉没有珍惜一个人的一辈子，莫名的负罪感。今天不能再这样了，我想通了，时间是这个世界上最公平的产物，当你花了时间在一样东西上时，必然对于另外一些来说是对时间的浪费。所以，这次一定要对自己负责，对自己的前途负责，好好对待时间，好好对待身边人，好好热爱自己的生活，本命年，也是最关键的一年，必须加油( _)
